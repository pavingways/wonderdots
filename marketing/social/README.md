# Social Media Posts

This folder contains social media post drafts for Wonder Dots. Posts are created here for review before publishing.

## Folder Structure

```
marketing/
  social/
    linkedin/     # LinkedIn posts (longer, professional tone)
    instagram/    # Instagram captions + hashtags + visual notes
    x/            # X/Twitter posts (short, punchy; often multiple options)
    tiktok/       # TikTok scripts/ideas (when applicable)
```

## Naming Schema

```
YYYY-MM-DD_slug.md
```

- **Date** = intended publish date (matches the related website article if applicable)
- **Slug** = matches the slug of the corresponding website article, or a short descriptive name for standalone posts

**Examples:**
- `2026-05-14_calliope-gamekit-handheld-spiel.md`
- `2026-05-21_rueckblick-kurs-meggen.md`
- `2026-06-01_neuer-kurs-ankuendigung.md`

## Post Types

| Type | Description |
|------|-------------|
| 📰 Article companion | Accompanies a new website article — references and links to it |
| 📣 Announcement | New course, event, or registration opening |
| 🔁 Retrospective | Summary/highlights of a past course or event |
| 💡 Standalone | General topic post (computational thinking, MINT education, etc.) |

## Workflow

1. Posts are drafted here (by the AI agent or a team member) with status `Entwurf – zur Freigabe`
2. Team reviews the draft in the PR
3. On approval: copy-paste to the platform and publish
4. Update the status in the file to `Veröffentlicht` with the actual publish date
